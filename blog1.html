Arrow Functions in JavaScript


Arrow functions are similar to anonymous functions in Java i.e functions which do not have any name


Example of Arrow functions :

// Regular Functions

function hello() { return 'hello world' }


// Arrow Functions

let hello = () => 'hello world'


Both functions work in the same way, even though there syntax is different. The () brackets represents the syntax for arguments.

We can skip the empty brackets if there is only one argument present in he function, like below:


var cube = num => num * num * num;


In one line arrow functions we can skip the return statement. We can also declare multiline arrow function by using culy {} brackets.


var cube = num =>{ return num * num * num }


Arrow functions with parameters:


var hello = (value) => "hello" + val;


Arrow functions with multiple parameters:


(p1, p2, ..., pn) => expression;
